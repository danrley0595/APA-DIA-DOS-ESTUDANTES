# APA Estudantes — Ciência de Dados

Projeto desenvolvido na Escola do Futuro do Estado de Goiás José Luiz Bittencourt,
como Atividade Prático-Acadêmica (APA) integrada entre os cursos de
Ciência de Dados, Desenvolvimento Web e Mobile e Marketing e Mídias Sociais.

## Tema

Drogas Ilícitas nas Escolas Brasileiras — análise baseada na PeNSE 2024 (IBGE).

## Tecnologias

- HTML5, CSS3, JavaScript (Vanilla)
- Fonte de dados: PeNSE 2024 — IBGE / Ministério da Saúde

## Funcionalidades

- Visualização de gráficos e análises estatísticas
- Carrossel de campanha de conscientização (Marketing)
- Modo escuro
- Menu responsivo com hamburguer
- Animações de scroll reveal
